# 🏥 AI Healthcare Appointment Management System

An AI-powered Healthcare Appointment Management System built using **ServiceNow App Engine Studio** to automate appointment booking, patient management, workflow automation, dashboards, and email notifications.

---

## ✨ Features

- 🤖 AI-based symptom severity classification
- 📅 Automated appointment scheduling
- 📧 Automatic email notifications
- 👨‍⚕️ Doctor recommendation system
- 🚨 Emergency prioritization
- 📊 Interactive Platform Analytics dashboards
- 🔄 Workflow automation using Flow Designer
- 🏥 Patient, Doctor & Appointment management
